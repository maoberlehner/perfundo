What?

perfundo is a pure CSS Lightbox which is completely functional without JavaScript.

Why?

First and foremost I just really like doing things, which are usually done with JavaScript, without JavaScript. Next my focus was to keep the file size of the CSS file (which provides the core functionallity) at an absolute minimum. So it is possible to provide a basic Lightbox functionallity for minimalistic sites with a focus on loading time and mobile performance.

Usually my answer why things should work also without JavaScript is: progressive enhancement. But in the case of Lightboxes progressive enhancement isn't an issue. The fallback of traditional Lightbox solutions is to just link to the big version of the image, which is a very good fallback behaviour in my opinion. Read more about why progressive enhancement is important.

How?

The short answer is: I use the CSS :target pseudo class. If you want to go into detail, I tried my best to document the code with comments.

animate.css is used for animations.
